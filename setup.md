For this workshop you have a [USQ Software Carpentry with R VM login](https://softwarecarpentry19.usq-eresearch-services-sandbox.cloud.edu.au/rstudio/) that has been created for you. Your username and password = your last name and first intial in lower case with no space. For example, for a user called Joe Bloggs, the username and password would be "bloggsj".

We will be using RStudio server in your web browser. RStudio is an application (an integrated development environment or IDE) that facilitates the use of R and offers a number of nice additional features.

If you wish to install RStudio on your computer after the workshop to keep working, it can be downloaded for free [here](https://www.rstudio.com/products/rstudio/download/). You will need the free "Desktop" version for your computer.
